# 타이포그래피 Semantic 토큰

Semantic 타이포그래피 토큰은 디자인 시스템에서 사용되는 텍스트 스타일을 용도에 맞게 정의한 토큰입니다. 이 토큰들은 디자인 시스템 전반에 걸쳐 일관된 텍스트 스타일을 유지하는 데 사용됩니다.
