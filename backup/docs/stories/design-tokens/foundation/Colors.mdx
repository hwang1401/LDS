## 사용 가이드라인

Foundation 색상 토큰은 직접 사용보다는 Semantic 토큰을 통해 사용하는 것이 권장됩니다. 이는 디자인 변경 시 일관성을 유지하고 유지보수를 용이하게 합니다.

```css
/* 권장하지 않음 */
.element {
  color: var(--foundation-color-blue-light-50);
}

/* 권장 */
.element {
  color: var(--semantic-color-primary-foreground-1-rest);
}
```
