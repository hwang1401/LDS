## 사용 가이드라인

Foundation 높이 토큰은 직접 사용보다는 Semantic 토큰을 통해 사용하는 것이 권장됩니다. 이는 디자인 변경 시 일관성을 유지하고 유지보수를 용이하게 합니다.

```css
/* 권장하지 않음 */
.button {
  height: var(--foundation-height-360);
}

/* 권장 */
.button {
  height: var(--semantic-size-button-md);
}
```

효과적인 높이 사용을 위한 원칙:

- **일관성**: 동일한 유형의 요소에는 동일한 높이 적용
- **접근성**: 충분한 터치/클릭 영역 확보 (모바일 요소는 최소 44px 권장)
- **비례**: 화면 크기와 요소의 중요도에 비례하는 높이 선택
- **정렬**: 높이가 다른 요소들 간의 적절한 수직 정렬 유지
