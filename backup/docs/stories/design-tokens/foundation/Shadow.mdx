# 그림자 Foundation 토큰

Foundation 그림자 토큰은 디자인 시스템에서 요소의 깊이감과 공간감을 표현하는데 사용되는 그림자 효과를 정의합니다. 이 토큰들은
transparent 색상과 함께 주로 계층 구조와 상호작용성을 시각적으로 표현하는 데 사용됩니다.

## 사용 가이드라인

Foundation 그림자 값은 직접 사용보다는 Semantic 토큰을 통해 사용하는 것이 권장됩니다. 이는 디자인 변경 시 일관성을 유지하고 유지보수를 용이하게 합니다.

```css
/* 권장하지 않음 */
.card {
  box-shadow: 0px 4px 6px var(--foundation-color-transparent-light-300);
}

/* 권장 */
.card {
  box-shadow: var(--semantic-shadow-component-resting);
}
```

효과적인 그림자 사용을 위한 원칙:

- **일관성**: 동일한 깊이와 중요도를 가진 요소에는 동일한 그림자 적용
- **계층 구조**: 그림자 강도를 통해 시각적 계층 구조 표현
- **상호작용**: 상태 변화(휴면, 호버, 활성)에 따라 적절한 그림자 전환
- **성능 고려**: 과도한 그림자 효과는 렌더링 성능에 영향을 줄 수 있으므로 필요한 곳에만 사용
