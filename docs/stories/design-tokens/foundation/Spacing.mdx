## 사용 가이드라인

Foundation 간격 토큰은 직접 사용보다는 Semantic 토큰을 통해 사용하는 것이 권장됩니다. 이는 디자인 변경 시 일관성을 유지하고 유지보수를 용이하게 합니다.

```css
/* 권장하지 않음 */
.card {
  margin-bottom: var(--foundation-spacing-160-vertical);
}

/* 권장 */
.card {
  margin-bottom: var(--semantic-spacingVer-global-md);
}
```

일관된 간격 시스템을 사용하면 다음과 같은 이점이 있습니다:

- **시각적 리듬**: 일관된 간격으로 UI 전체에 안정적인 시각적 리듬 제공
- **반응형 디자인**: 정의된 간격 토큰을 통해 다양한 화면 크기에 따라 일관된 비율로 조정 가능
- **유지보수성**: 전체 시스템의 간격을 쉽게 조정 가능
