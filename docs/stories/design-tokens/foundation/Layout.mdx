# 레이아웃 Foundation 토큰

Foundation 레이아웃 토큰은 디자인 시스템의 기본적인 크기와 간격 단위를 정의합니다. 이 토큰들은 시맨틱 토큰의 기반이 되며, 디자인 시스템 전반에 걸쳐 일관된 간격과 크기를 유지하는 데 사용됩니다.

## 간격 (Spacing)

간격 토큰은 UI 요소 간의 간격이나 패딩, 마진 등을 정의하는 기본 단위입니다.
