# 형태 Foundation 토큰

Foundation 형태 토큰은 디자인 시스템의 기본적인 모양과 형태를 정의합니다. 이 토큰들은 시맨틱 토큰의 기반이 되며, 디자인 시스템 전반에 걸쳐 일관된 시각적 요소를 유지하는 데 사용됩니다.
