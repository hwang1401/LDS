# 폰트 Foundation 토큰

Foundation 폰트 토큰은 디자인 시스템의 기본 텍스트 스타일을 정의합니다. 이 토큰들은 직접 사용되기보다 Semantic 폰트 토큰을 통해 참조됩니다.

## 폰트 패밀리

    borderRadius: '4px',
    marginBottom: '8px'
  }}>
    <div style={{ fontSize: '20px', marginBottom: '8px' }}>Pretendard</div>

## 사용 가이드라인

Foundation 폰트 토큰은 직접 사용보다는 Semantic 토큰을 통해 사용하는 것이 권장됩니다. 이는 디자인 변경 시 일관성을 유지하고 유지보수를 용이하게 합니다.

```css
/* 권장하지 않음 */
.element {
  font-size: var(--foundation-typography-fontSize-160);
}

/* 권장 */
.element {
  font-size: var(--semantic-typography-body-1-fontSize);
}
```
