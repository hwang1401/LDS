# 형태 Semantic 토큰

Semantic 형태 토큰은 디자인 시스템에서 사용되는 UI 요소의 형태적 특성을 정의하는 토큰입니다. 이 토큰들은 디자인 시스템 전반에 걸쳐 일관된 시각적 언어를 제공합니다.
