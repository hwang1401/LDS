# 레이아웃 Semantic 토큰

Semantic 레이아웃 토큰은 디자인 시스템에서 사용되는 요소의 배치와 구조를 정의하는 토큰입니다. 이 토큰들은 디자인 시스템의 일관된 레이아웃 패턴을 유지하는 데 사용됩니다.
