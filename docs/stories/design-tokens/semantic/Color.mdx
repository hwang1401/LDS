# 색상 Semantic 토큰

Semantic 색상 토큰은 디자인 시스템에서 사용되는 색상의 의미와 용도를 정의하는 토큰입니다. 이 토큰들은 디자인 시스템의 일관된 색상 언어를 유지하는 데 사용됩니다.

각 색상 시맨틱 토큰은 다음과 같은 4단계 계층 구조를 따릅니다:

```
상황.용도.위계.상태
```

1. **상황(Context)**: 사용되는 맥락 - primary, secondary, cta, status 등
2. **용도(Usage)**: 실제 활용 목적 - background, foreground, stroke, oncolor 등
3. **위계(Hierarchy)**: 중요도 수준 - 1(가장 강조), 2(중간), 3(낮은 강조) 등
4. **상태(State)**: CSS pseudo-class 기반 상태 - rest, hovered, pressed, selected, disabled 등

## 브랜드 색상 (Primary)
